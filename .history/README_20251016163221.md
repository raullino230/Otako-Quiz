# Site Lofi

Pequeno site Flask com quiz. Instruções para publicar no GitHub Pages (conteúdo estático em `templates/` e `static/`).

Como executar localmente:

1. Criar um ambiente virtual

```powershell
python -m venv venv; .\venv\Scripts\Activate.ps1; pip install -r requirements.txt
```

2. Rodar a aplicação

```powershell
python main.py
```

Publicar no GitHub (GitHub Pages):

- Se preferir usar GitHub Pages com conteúdo estático, você pode gerar uma versão estática (exportar `templates/site.html` e `static/` para a raiz do branch `gh-pages`) ou hospedar como página estática no branch `gh-pages`.
- Alternativamente, para manter a app Flask em execução, use um serviço como Railway, Render, Heroku ou uma VM com suporte a WSGI.
